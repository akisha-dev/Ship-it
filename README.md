# Ship it or Skip it 

A fun React app that helps you decide which project ideas to build next.

## Features
- Browse through project ideas one by one
- Ship it to save ideas you want to build
- Skip it to move to the next idea
- See your saved ideas in a shipped list

## Built With
- React
- Tailwind CSS
- Vite

## What I Learned
- React useState hook
- Component reusability with props and children
- Conditional rendering with ternary operators
- Tailwind CSS utility classes
